# 📖 Descripción

**Raccoon Cash** es una aplicación móvil desarrollada en Android cuyo objetivo es ayudar a los usuarios a administrar sus finanzas personales de forma sencilla, organizada y eficiente.

La aplicación permite registrar ingresos, controlar gastos, administrar múltiples cuentas financieras y visualizar resúmenes mensuales para facilitar la toma de decisiones económicas.

El proyecto fue desarrollado aplicando principios de **Programación Orientada a Objetos (POO)** y una arquitectura moderna basada en **MVVM (Model - View - ViewModel)**.

# 🧠 Conceptos de Programación Orientada a Objetos implementados

### Encapsulamiento

Protección y organización de los datos mediante clases y propiedades.

### Abstracción

Separación de la lógica de negocio y la interfaz de usuario.

### Herencia

Reutilización de comportamientos cuando sea necesario.

### Polimorfismo

Flexibilidad en la implementación de componentes.
